overfloat 属性
- visible 默认值,内容不会被修剪,会呈现在元素之外
- hidden 内容会被修剪,并且其余的内容是不可见
- scrol 内容会被修剪,但是浏览器会显示滚动条以查看其余的内容
- auto 如果内容被修剪,则浏览器显示滚动条查看其余内容
- inherit 从父元素继承overfloat的属性的值
