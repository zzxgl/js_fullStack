  em 移动端少用Px rem 细致的每个盒子或盒子的模型的大小等
  比例定义
  em 是相对于自身的字体的大小来比例
  10em 10*14px=140px
  rem html 根元素的fontsize
  ：before :after 为元素
  没有标签，但却可以像真正的元素一样，来在页面上占位置
  dom 文档流中不需要写这个元素，副作用，不会影响到内容的显示
  使用CSS来声明，一定要有content属性是必需的，

  HTML 默认的字体大小是16px
  如果没有设置，会使用父级， 或 body font-size
